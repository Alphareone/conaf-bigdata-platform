📖 Descripción General

Este proyecto consiste en el diseño e implementación de una plataforma analítica basada en tecnologías Big Data para apoyar la gestión de emergencias forestales en Chile.

La solución integra información proveniente de incendios forestales históricos, infraestructura crítica de salud y variables meteorológicas, permitiendo generar análisis de riesgo, visualizaciones geográficas y apoyo a la toma de decisiones para organismos relacionados con la prevención y gestión de emergencias.

El proyecto fue desarrollado como actividad semestral de la asignatura Big Data, aplicando conceptos de arquitectura de datos, procesamiento ETL, analítica en la nube y visualización de información.

🎯 Objetivo del Proyecto

Desarrollar una plataforma de análisis de datos que permita:

Integrar múltiples fuentes de información.
Analizar eventos históricos de incendios forestales.
Incorporar información de establecimientos de salud.
Considerar variables meteorológicas para el análisis de riesgo.
Facilitar la visualización y exploración de datos mediante dashboards interactivos.
Aplicar herramientas de procesamiento y análisis de datos en entornos cloud.
🌎 Problemática

Los incendios forestales representan una amenaza constante para las comunidades y la infraestructura crítica del país.

La disponibilidad de información dispersa dificulta la identificación temprana de riesgos y la planificación de respuestas oportunas.

Este proyecto busca centralizar y relacionar distintas fuentes de datos para proporcionar una visión más completa del impacto potencial de los incendios sobre zonas sensibles y establecimientos de salud.

🏗️ Arquitectura General

La solución fue diseñada siguiendo principios de arquitectura Big Data para:

Capturar información desde distintas fuentes.
Almacenar los datos en servicios cloud.
Realizar procesos de transformación y limpieza.
Consolidar información en un entorno analítico.
Generar visualizaciones e indicadores para la toma de decisiones.
☁️ Tecnologías Utilizadas
Google Cloud Platform (GCP)
Cloud Storage

Utilizado como repositorio central para el almacenamiento inicial de datasets.

Cloud Dataprep

Herramienta utilizada para procesos de limpieza, transformación y preparación de datos.

BigQuery

Motor de almacenamiento y análisis de datos utilizado para consultas analíticas de gran escala.

BigQuery ML

Aplicado para la exploración de modelos analíticos directamente sobre los datos almacenados.

Looker Studio

Utilizado para la construcción de dashboards e informes interactivos.

Este proyecto fue desarrollado utilizando un entorno temporal de laboratorio proporcionado por Google Cloud Platform.

Debido a las características del laboratorio académico:

Los recursos cloud tenían una duración limitada.
Parte de la infraestructura original ya no se encuentra disponible.
Algunos datasets, configuraciones y servicios fueron eliminados automáticamente al finalizar el período de acceso.
